    Learning Scala¶

    学习Scala的好书，博客和其他资源

    Books¶

    • Essential Scala Noel Welsh和Dave Gurnell撰写-Essential Scala的目标读者是第一次接触Scala的经验丰富的开发人员.
    • Functional and Reactive Domain Modeling 通过Debasish Ghosh
    • Functional Programming for Mortals 由山姆·哈利迪（Sam Halliday）
    • Functional Programming in Scala Paul Chiusano和Runar Bjarnason
    • Functional Programming, Simplified (Scala edition) 阿尔文·亚历山大（Alvin Alexander）
    • Get Programming with Scala Daniela Sfregola编写的-Scala的教程驱动简介
    • Practical FP in Scala: A hands-on approach 加布里埃尔·沃尔佩（Gabriel Volpe）
    • Programming in Scala 马丁·奥德斯基（Martin Odersky），莱克斯·斯普恩（Lex Spoon）和比尔·文纳斯（Bill Venners）
    • Reactive Programming with Scala and Akka -使用反应式编程的概念来构建在多个节点上运行的分布式系统
    • Scala Cookbook 阿尔文·亚历山大（Alvin Alexander）
    • Scala for the Impatient 作者：Cay Horstmann-涵盖了Scala的大多数功能，并提供了简短易懂的说明.
    • Scala With Cats 由Noel Welsh和Dave Gurnell撰写-使用现代函数式编程技术来学习系统架构和设计 Cats
    • The Type Astronaut's Guide to Shapeless 戴夫·古内尔（Dave Gurnell）
