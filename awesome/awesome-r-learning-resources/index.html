    Awesome r learning resources


    >“ Awesome R Learning Resources”资源库旨在帮助所有技能水平和背景的用户加深对R的理解，R是一种用于统计计算和图形的编程语言和环境.


    >RDiscord`服务器是一个面向R爱好者，程序员，统计学家，数据科学家和学生的友好而专用的社区. 无论您是想与其他用户联系，共享令人敬畏的数据，还是只是需要统计信息的帮助，您都来对地方了！

    要加入R Discord服务器，请单击下面的discoRd徽章.

    Topic Areas¶

    Comprehensive R Tutorials¶

    • Data Flair -这些教程按技能级别（初学者，中级，专家）分组.
    • R CODER -这些教程分为几类（简介，数据结构，数据整理，编程，导入和导出，图形），涵盖了开始学习R编程语言的人员的所有基本需求.
    • Tutorials Point -这些教程分为几类（R教程，R数据接口，R图表和图形，R统计示例，R有用资源），涵盖了开始学习R编程语言的人员的所有基本需求.

    Functions¶

    • stat.berkeley - Introduction to Functions -将计算集成到统计课程（UC Berkeley）的组织者对R语言功能的介绍.

    Generative Art¶

    • 12 Months of aRt -在2019年，威廉·蔡斯（William Chase）开始一个项目，每月制作完全由R制作的新作品系列.在该项目中，他探索了不同的技术，开发了算法并提供了详细的帖子，详细介绍了每个月的开发过程.

    Joining Data¶

    • Joining Data in R with dplyr -DataCamp上的“使用dplyr连接R中的数据”中的课程笔记. 主题包括变异联接，过滤联接和设置操作，组装数据，高级联接. 作者：威廉·萨尔斯（William Surles）.

    Math¶

    • Descriptive Statistics -描述性统计信息教程，用于以汇总数据的方式提供对数据中包含的信息的洞察力. 作者：Salvatore S. Mangiafico.
    • Essential Math for Data Science -讨论关键数学主题以掌握成为更好的数据科学家的文章. 作者：Tirthajyoti Sarkar.
    • Gallery of Statistical Distributions -作者：NIST / SEMATECH.
    • [Plotting distributions (ggplot2)](http://www.cookbook-r.com/Graphs/Plotting_distributions_(ggplot2) /）-用于绘制数据分布的教程. 作者：Winston Chang.

    Shiny¶

    • Awesome R Shiny -R Shiny的精选资源列表. 作者：Rob Gilmore.
    • Building Shiny Applications with R Tutorial (Deprecated) -Shiny入门教程. 请注意，本教程已弃用. 作者：RStudio.
    • Building Shiny apps - an interactive tutorial -本教程是对一系列实践活动的补充 presentation slides 学习如何构建Shiny应用程序. 作者：院长阿塔利（Dean Attali）.
    • How to Start with Shiny -详细的入门视频教程. 作者：加勒特·格罗莱蒙德（Garrett Grolemund）.
    • Learn Shiny -此页面上的视频和书面教程主要是针对Shiny的新手并且需要指导性介绍的用户而设计的. 作者：RStudio.
    • Shiny Articles -各种文章，涵盖了更高级的各个Shiny主题. 作者：RStudio.

    Spatial¶

    • An Introduction to Choropleth maps in R -作者：亨利·坎恩.
    • Getting latitude & longitude for any address -作者：Brecht Vermeire.
    • Map Plots Created With R And Ggmap -作者：劳拉·埃利斯（Laura Ellis）.
    • Plot Spatial Data / Shapefiles in R -来自“ math et al” YouTube频道.

    Viz¶

    • A ggplot2 Tutorial for Beautiful Plotting in R - A comprehensive and easy to follow tutorial that covers working with axes, titles, legends, backgrounds, grid lines, margins, multi-panel plots, colors, themes, lines, text, coordinates, chart types, ribbons, smoothings, and interactive plots. Author: Cédric Scherer.
    • Awesome ggplot2 -精选的ggplot2教程，软件包等的精选列表.作者：Erik Gahner Larsen.
    • Chart Suggestions — A thought-starter on choosing the way to show your data -作者：安德鲁·阿贝拉（Andrew Abela）博士
    • Color Hex Color Codes -作者：Color-Hex.
    • Combine Multiple GGPlots into a Figure -作者：Alboukadel Kassambara.
    • From Data to Viz -从数据到可视化将引导您找到最适合您数据的图形. 作者：严·霍尔兹（Yan Holtz）.
    • ggplot2 extensions - gallery -由Daniel Emaasit维护.
    • ggplot2 - Modify components of a theme -如何在ggplot2中修改主题的组件. 作者：Tidyverse的开发人员.
    • Graphics in R with ggplot2 -有关在ggplot2中使用图形的详细指南. 作者：Antoine Soetewey.
    • htmlwidgets for R -您可以使用R构建的各种交互式Web可视化的展示和画廊.
    • r-color-palettes -r中提供的调色板的完整列表. 作者：埃米尔·赫维特费尔特（Emil Hvitfeldt）.
    • The Data Visualization Catalogue -数据可视化目录是Severino Ribecca开发的一个项目，用于创建不同信息可视化类型的库.
    • The Graphic Continuum -图形连续体显示了我们编码和显示数据时可以使用的许多不同类型的可视化. 作者：乔纳森·施瓦比什（Jonathan Schwabish）和塞韦里诺·里贝卡（Severino Ribecca）.
    • The R Graph Gallery -用R编程语言制作的图表集合. 作者：严·霍尔兹（Yan Holtz）.
    • Time Based Heatmaps in R -作者：劳拉·埃利斯（Laura Ellis）.
    • Top 50 ggplot2 Visualizations - The Master List (With Full R Code) -本教程可帮助您选择适合特定目标的图表类型，以及如何使用ggplot2在R中实现图表. 作者：塞尔瓦·普拉巴卡兰（Selva Prabhakaran）.

    Wrangling¶

    • Data manipulation in r using data frames - an extensive article of basics -作者：dataENQ.
    • Data Wrangling Part 1: Basic to Advanced Ways to Select Columns -作者：苏赞·贝特（Suzan Baert）.

    • Data Wrangling Part 4: Summarizing and slicing your data -作者：苏赞·贝特（Suzan Baert）.

    • data.table: A gentle introduction -快速介绍data.table. 主要目的是介绍data.table语法，显示如何执行基本但必不可少的数据整理任务. 作者：Atrebas.
    • Fakir - Create Fake Data in R for Tutorials -作者：科林·费伊（Colin Fay）.

    • Alex Cookson -Alex Cookson喜欢对R概念进行漂亮的可视化和易于阅读的演练. 他对书籍，电影和音乐剧等媒体数据特别感兴趣.
    • Avery Robbins -艾利·罗宾斯（Avery Robbins）热爱学习和分享有益于他的个人有益或令人敬畏的事物. 该网站是他积极从事此工作的工具：分享有用的知识，想法和技巧.
    • Tony ElHabr -Tony ElHabr对能源市场和体育分析充满热情. 他的博客提供了详细的教程，项目说明和演示.
    • Cédric Scherer -CédricScherer是一名经过培训的计算生态学家，并且是一位心地善良的数据可视化设计师，具有9年以上的假设驱动研究经验，并且在数据整理，统计分析，模型开发和数据可视化方面具有很强的技能.
    • Data Imaginist -Thomas Lin Pedersen是一名由数据科学家转为软件工程师，致力于改善研究人员与其产生的数据的交互.
    • Data meets Narrative -丽贝卡·易货（Rebecca Barter）喜欢理解复杂的，混乱的，有时甚至是无意义的数据集，例如电子健康记录和保险索赔. 她的双重热情是用通俗的英语向其他人解释“看似复杂”的概念，并探索和发现构成复杂数据集的故事.
    • HighlandR -约翰·麦金托什（John Mackintosh）的博客是一个展示演示或研讨会的地方，他在工作中学到的知识，图表改造以及他目前不使用的技术和技巧.
    • Julia Silge -朱莉娅·席尔格（Julia Silge）是RStudio的数据科学家和软件工程师，她在研究开源建模工具. 她热衷于制作精美的图表，统计编程语言R，简·奥斯丁，黑咖啡和红酒.
    • Musings on R -陈（Martin Chan）撰写的有关R和数据科学万物的博客. 涵盖的主题包括比较dplyr和data.table，Shiny应用程序，ggplot，数据清理，使用RStudio，与其他R用户/数据科学家的访谈以及网络抓取.
    • rweekly -整个R社区的每周更新，包括Bruce Zhao，Colin Fay，Eric Nantz，Hao Zhu，Jon Calder，Jonathan Carroll，MaëlleSalmon，Ryo Nakagawara和Wolfram Qin.
    • r-bloggers -R-Bloggers.com由Tal Galili创建，是博客撰写者的博客内容，这些博客作者撰写了有关R（英文）的文章. 该站点可帮助R博客作者和用户建立联系并关注R博客圈.
    • Ryo Nakagawara -Ryo Nakagawara是一名数据科学家，并且一直作为报告分析师以及R和SQL的软件开发人员来改善ACDI和VOCA数据管道，创建R包，可再现的报告，仪表板和Shiny应用程序，以交流他的项目的工作方式全球范围内都在进步.
    • Statistics Globe -Joachim Schork启动了该平台，与其他统计学家和程序员进行讨论，以分享他的统计知识并提高他自己的统计技能.
    • Stats and R -通过他的博客，Antoine Soetewey（统计博士学位）旨在帮助研究数据的学者和专业人员掌握重要的统计概念，并展示如何在R中应用它们.

    Books¶

    • A Sufficient Introduction to R -本书旨在指导那些完全不熟悉R的人员使用R.作者：Derek L. Sonderegger.
    • An Introduction to Statistical Learning -本书介绍了统计学习方法. 作者：Gareth James，Daniela Witten，Trevor Hastie和Robert Tibshirani.
