    Awesome cdk

    >精选列表很棒 AWS Cloud Development Kit （AWS CDK）开源项目，指南，博客和其他资源.

    AWS Cloud Development Kit（AWS CDK）是一个开源软件开发框架，用于在代码中定义云基础架构.

    Construct Libraries¶

    本节包含各种编程语言的代码库，这些代码库提供可在CDK应用程序中使用的构造.

    APIs¶

    • cdk-chalice -适用于AWS Chalice（适用于AWS的Python无服务器微框架）的AWS CDK构造.
    • auto-cdk -自动生成与文件系统（beta）的api-gateway / lambda集成.

    Databases¶

    • aws-cdk-dynamodb-seeder -用于DynamoDB的简单CDK播种器.
    • cdk-tweet-sentiment -识别推文中的情感，并将其记录到Amazon DynamoDB表中.
    • cdk-postgresql -适用于PostgreSQL的AWS CDK构造.
    • cdk-sqlserver-seeder -对SQL Server数据库执行自定义SQL脚本的CDK构造.

    Static Websites¶

    • cdk-static-website -CDK组件，可使用S3创建静态网站，配置CloudFront（CDN）并通过Route53（DNS）映射自定义域.
    • ness -CDK驱动的CLI工具，用于将静态站点部署到您的AWS账户.

    Security¶

    • cdk-passwordless -使用userpool进行无密码身份验证的构造.
    • cdk-iam-generator - Construct to generate IAM Managed Policies and IAM Roles using JSON Configuration.
    • c3 -符合隐私和安全最佳做法.
    • cdk-iam-floyd -具有流畅接口的IAM策略声明生成器.
    • k9-cdk -轻松生成安全的S3存储桶策略.

    Ops¶

    • cdk-instanceStopRule -CDK组件，该组件使用CloudWatch规则创建实例以在一天结束时将其停止.
    • cdk-time-bomb -CDK构造，在一定时间后会爆破您的AWS CDK堆栈.

    Queue¶

    • cdk-tweet-queue -用高音扬声器搜索查询的高音填充SQS队列.
    • cdk-ses-template-mailer -用于创建AWS SES电子邮件模板+微服务的构造，以使用AWS SES发送模板化电子邮件.
    • cdk-sqs-monitored -具有死信队列和已配置警报的SQS构造.

    CI/CD¶

    • aws-delivlib -合成CI / CD管道以交付多语言软件（由CDK本身使用）.
    • cdk-blue-green-container-deployment -使用CodeDeploy进行蓝色/绿色容器部署.

    Monitoring¶

    • cdk-watchful -CDK应用程序的自动仪表板和警报.
    • aws-cdk-billing-alarm -构造用于设置电子邮件警报的金额超过您的AWS账单上的金额.

    Workflows¶

    • cdk-pull-request-check -CDK组件，可自动检查拉取请求.
    • cdk-github-webhook -提供GitHub webhooks的CDK组件.
    • cdk-codepipeline-slack - CDK component that provisions a #slack approval workflow.
    • cdk-codecommit-backup -将CodeCommit存储库备份到S3.
    • Alexa Deployment Pipeline -使用AWS SAM和DeployToAlexa操作创建一个CodePipeline来将Alexa Skills部署到Lambda和开发人员控制台的构造.
    • cdk-developer-tools-notifications -开发人员工具的Slack / Microsoft团队/电子邮件通知：CodeCommit，CodeBuild，CodeDeploy，CodePipeline.

    High-Level Frameworks¶

    • punchcard -TypeScript框架，用于统一CDK的基础结构和运行时代码，因此您可以在一个Node.js应用程序的上下文中声明构造并实现运行时逻辑.
    • aws-cdk-pure -使用AWS CDK开发纯功能和高级云组件的工具包.
    • cdk-stepfunctions-patterns -一组“阶梯功能”高级弹性模式.

    Scaffolding¶

    • ReactJS + Cognito + CDK Starter -具有AWS CDK支持的ReactJS + Amazon Cognito + Amazon Amplify Framework的入门项目.
    • cra-template-aws-cdk -使用AWS CDK开箱即用，简单地配置无服务器React应用程序即可创建React App模板.
    • create-cdk-app -从模板创建CDK应用程序.
    • awscdk-jsii-template -GitHub模板存储库，可生成一个就绪的环境来构建，测试和发布您 [JSII](https://github.com/kolomied/awesome-cdk/blob/master/(https://github.com/aws/jsii)）为AWS CDK构建lib.

    Language Support¶

    • AWS-CDK-Kotlin-DSL -的包装器库 AWS CDK Java . CI自动生成代码，并每天进行部署.
    • aws-cdk-maven-plugin -使用Java和Maven定义和部署AWS CDK应用程序的插件.
    • aws-lambda-nodejs-webpack -使用以下替代Node.js lambda CDK构造 webpack.
    • aws-lambda-nodejs-esbuild -使用以下替代Node.js lambda CDK构造 esbuild.

    Library Publishing¶

    • GitHub Action -适用于AWS CDK的GitHub Action.
    • jsii-publish - 一种 Docker image 和 GitHub action to build 和 publish CDK constructs created via JSII.
