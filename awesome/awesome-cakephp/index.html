    CakePHP

    Awesome CakePHP ¶

    精选的令人赞叹的 CakePHP 3.x + 插件，资源和闪亮的清单.

    带有“：strawberry：”图标的插件也具有CakePHP 4兼容版本.

    如果您正在寻找CakePHP 2.x资源，请访问： - CakePHP 2.x version 这个很棒的清单 -这个维基 list of not-yet upgraded plugins

    您可能会发现有用的其他列表： - CakePHP Plugins - Awesome PHP - Awesome Awesomeness

    >对于那些想知道的人； 该列表与plugins.cakephp.org的不同之处在于支持 >插件子部分（而不是整个插件/仓库），更细化 >分组，并且主要关注于特定于任务的功能.

    Plugins¶

    APM¶

    用于应用程序性能监视的插件.

    • NewRelic plugin -一个完整的插件，可为CakePHP应用程序实现完整的New Relic集成，包括CLI命名，异常发送，自定义计时等.
    • NewRelic plugin -一个简单的插件，可以使用New Relic PHP代理仅进行名称交易和浏览器计时.

    Architecture¶

    • ：草莓： Burzum/CakeServiceLayer plugin -服务层和域/业务模型的实现.

    Asset Management¶

    用于管理，压缩和最小化网站资产的工具.

    • ：草莓： AssetCompress plugin -CakePHP的完整资产管理器.
    • ：草莓： AssetMix plugin -提供与 Laravel Mix 资产编制.
    • Assets plugin -动态和“即时”资产文件.
    • Less plugin -CakePHP的解析器插件更少.
    • MinifyHtml plugin -压缩HTML输出.

    Auditing / Logging¶

    用于审核和日志记录的插件.

    • AuditStash plugin -灵活可靠的审核日志跟踪.
    • ：草莓： DatabaseLog plugin -简单而独立的日志记录到数据库而不是文件.
    • ：草莓： Muffin/Footprint plugin -插件以允许将当前登录的用户传递到模型层.
    • Version plugin -有助于版本化数据库实体的插件.

    Authentication and Authorization¶

    *用于实现身份验证和授权的插件和库.

    • ：草莓： Acl plugin -将ACL作为数据库方法进行管理.
    • ADmad/HybridAuth plugin -允许使用 HybridAuth CakePHP在图书馆的社交标志.
    • ADmad/JwtAuth plugin -用于使用JSON Web令牌进行身份验证的插件.
    • ：草莓： Authentication plugin -官方的CakePHP身份验证中间件插件.
    • ：草莓： Authorization plugin -官方CakePHP授权栈.
    • CakeDC/NavAuth plugin -使用SOAP或OData服务针对Navision®服务进行身份验证的插件. 它包括NTLM身份验证等.
    • ：草莓： CakeDC/Users plugin -完整的用户管理（管理面板，记住我等），社交登录（FB，Twitter，LinkedIn，Google，Instagram），RBAC，API等.
    • CookieAuth plugin -一个简单的Cake 3插件，可通过Cookie自动验证用户身份.
    • HierAuth plugin -一个CakePHP插件，用于分层，基于角色的简单授权.
    • Muffin/OAuth2 plugin -使用OAuth2身份验证 league/oauth2-client.
    • ：草莓： Muffin/Tokenize plugin -事件驱动的行为，可轻松生成一次性安全令牌.
    • MultiTenant plugin -轻松构建支持SaaS的Web应用程序.
    • ：草莓： TinyAuth plugin -基于身份验证和基于角色的（单/多）授权，这是一种非常轻巧的方法.
    • ：草莓： Tools:Passwordable -包含 Passwordable behavior 用于密码哈希的DRY方法.
    • ：草莓： TwoFactorAuth plugin -允许使用Google Authenticator或类似应用进行两因素身份验证，以生成一次性代码. 基于 RobThree/TwoFactorAuth 图书馆.
    • UserPermissions plugin -允许用户组或单个用户查看特定页面.

    Caching¶

    *用于缓存数据的插件.

    • ：草莓： Cache plugin -用于将视图（HTML，CSV，JSON，XML等）作为静态缓存文件进行缓存.

    Code Analysis¶

    *用于分析，解析和处理代码库的插件.

    • ：草莓： IdeHelper plugin -通过将注释添加到现有代码中，类似于对新代码进行烘烤，从而有助于更好地支持IDE.
    • ：草莓： TestHelper plugin -提供测试增强功能和TDD支持作为浏览器后端.

    Debugging¶

    用于调试的插件.

    • Airbrake plugin 一个将Airbrake与CakePHP无缝集成以解决错误和异常的插件.
    • ：草莓： DebugKit plugin -用于调试的实际标准.
    • ErrorEmail plugin -一个将异常/错误信息通过电子邮件发送给您的开发团队的插件.
    • ：草莓： Execution order -一个演示应用程序，用于显示文件，方法和回调的执行顺序.
    • Psa/FixtureCheck plugin -一个插件，可帮助检测实时数据库和夹具中的不匹配情况，从而使基于夹具的测试更加可靠，并且部署更安全.
    • ：草莓： Sentry plugin 一个将Sentry与CakePHP无缝集成以解决错误和异常的插件.
    • ：草莓： Setup plugin -包含调试和维护工具的轻量级安装插件.
    • Whoops plugin -酷孩子的PHP错误和例外 filp/whoops.

    Dependency Injection¶

    实现依赖项注入设计模式的插件.

    • PimpleDi plugin 允许基于Pimple库的依赖项注入.
    • PipingBag plugin -依赖注入容器插件，增加了在使用对象实例及其依赖项之前对其进行配置的功能，并将其存储到容器类中以便于访问.

    E-commerce¶

    用于付款和建立在线电子商务商店的插件和应用程序.

    • PaypalWPP plugin -用于与Paypal Web Payments Pro通信以获取有关您帐户的交易和信息.

    Email¶

    用于发送和解析电子邮件的插件.

    • Elastic Email plugin -电子邮件传输插件，用于通过Elastic Email API发送电子邮件.
    • ：草莓： EmailQueue plugin -具有预览和发件人外壳的电子邮件队列插件.
    • Gourmet/Email plugin -电子邮件帮助程序，布局等.
    • ：草莓： Mailgun plugin -电子邮件传输插件，用于通过Mailgun发送电子邮件.
    • SendGrid plugin -电子邮件传输插件，用于通过SendGrid API发送电子邮件.

    Environment¶

    用于环境的插件.

    • Environments plugin -用于处理环境的插件.
    • Gourmet/Aroma plugin -基于数据库的配置.
    • Settings plugin -一个通过数据库管理您的设置的插件.

    File Manipulation¶

    用于文件操作的插件.

    • ：草莓： FileStorage plugin -抽象文件存储和上传插件.
    • FlyPie plugin -使用Flysystem进行抽象文件系统访问.
    • Image plugin -图像行为与Cake在TranslateBehavior中内置的行为非常相似.
    • ：草莓： Josbeir/Filesystem plugin -摘要 Flysystem +基于文件实体的上传插件.
    • ：草莓： Josegonzalez/Upload plugin -使用的可自定义插件 Flysystem 写入多个后端（Dropbox，FTP，S3，本地等）.
    • Proffer plugin -具有缩略图生成功能的可自定义上传插件.
    • Xety/Cake3Upload plugin -一个用于上传文件的小插件.

    Filtering and Validation¶

    用于过滤和验证数据的插件.

    • Gourmet/Filters plugin -额外的调度程序过滤器（维护，机械手，IP等）.
    • Gourmet/Validation plugin -额外的验证提供程序（Respect，IsoCode等）和规则.
    • HtmlPurifier plugin -具有特征，行为和助手的Purifier插件，可让您在需要的地方进行清理和过滤. 您也可以配置多组过滤规则.
    • HtmlPurifier plugin -净化器插件行为，用于在将数据封送到实体中和/或保存之前清除数据.

    Geolocation¶

    用于对地址进行地理编码以及使用纬度和经度的插件.

    • ：草莓： Geo plugin -包含 Geocoder behavior 和 GoogleMaps helper.

    HTTP¶

    用于HTTP和客户端抽象的插件

    • ：草莓： Http/Adapter/Cake library -适配器 HTTPlug HTTP客户端抽象.

    I18n¶

    • I18n（国际化）和L10n（本地化）的插件.

    • ：草莓： ADmad/I18n plugin -具有I18n相关工具的插件.

    • ：草莓： Cake/Localized plugin -本地化验证和即用型翻译PO文件.
    • ：草莓： ShadowTranslate plugin -一个基于影子表的插件，用于替换核心的Translate行为.
    • Transifex plugin -通过Transifex API管理i18n PO文件和翻译.
    • Translate plugin -通过网络后端（包括）轻松地管理静态内容的翻译. 从POT文件导入，自动建议和通过API自动翻译.
    • Translation plugin -提取Pot文件，翻译字符串（通常是Google，社区），将翻译导出到Pot文件.

    Imagery¶

    用于处理图像的插件.

    • ：草莓： ADmad/Glide plugin -使用的插件 Glide 图像处理库.
    • HtmlToImageView plugin -使用渲染HTML视图为图像（jpg或png） wkhtmltoimage.
    • Imagine plugin -图像处理插件和包装器 Imagine.
    • Thumber plugin -使用以下工具创建缩略图的插件 intervention/image.

    Libs¶

    不属于任何其他类别的有用的库或工具.

    • Capcake -使用Capistrano部署CakePHP应用程序.
    • Chronos -一个简单的独立DateTime API扩展（Carbon的后继）.
    • Composer Installers -多框架Composer库安装程序.
    • Composer/Packagist -包和依赖项管理器.
    • Graphviz -Graphviz库.
    • Jenkins -私有（GitHub）仓库的免费替代方案.
    • Rocketeer -PHP任务运行程序和部署包.
    • Travis CI -持续集成平台-实际的测试标准（GitHub）存储库.
    • YamlRoute -使用简单的YAML文件配置路由.

    Markup¶

    用于标记的插件.

    • Gourmet/CommonMark plugin -添加 CommonMark Markdown解析.
    • ：草莓： Markup plugin -允许使用基于PHP或JS的语法突出显示.

    Migration¶

    *围绕迁移和升级的插件和资源.

    • ：草莓： Migrations plugin -（DB）迁移插件.
